## 프로젝트 진척상황 공유2 (minj0i) - 2020.08.30

### 프로젝트 진척상황

* 로직
  - 앱을 설치하면 그 날을 저장함
  - 로그인을 구현한다면 로그인한 날을 저장함
    - date형식으로 들어가야 날짜 비교가 될 듯 함
  - 기본 done false로 날짜 생성하면서 넣고, 버튼 클릭 시 done->true

```
//데이터 예시
{
  "TODO1" : {
    "2020-08-30": {
      "done": false
    },
    "2020-08-31" : {
      "done": false
    },
    ...
  },
  TODO2 : {

  }
}
```
