# ohbokdongProject1
그냥 연습 프로젝트

## 개요
앱에 온라인 광고를 기재하여 앱 스토어에 앱을 올려 상용화 진행

상용 경험 및 광고 기능 추가에 대한 방법 학습에 중점

앱의 사용 정보를 수집하여 서버 운용 경험 누적

### 목적 
- 상용 경험 누적 (구글 플레이 스토어 출시)
- 온라인 광고 적용 (google Ads 사용)
- 서버 제작 및 데이터 베이스 활용 경험 (채팅 서버 및 정보 수집 DB 구축)


### 수집 데이터 목록
- 사용 단말기 정보
- 접속 횟수 
- 광고 정보
- 광고 진행 횟수
