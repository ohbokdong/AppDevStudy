# selfCoin

## 개요
- 브랜딩이  자신의 성장 가치를 스스로 측정하는 앱
- 개인의 목표를 설정하고, 성장 과정을 To do list로 체크
- 매일의 목표 설정을 성취하면 성장세 / 성취하지 못하면 하락세로 표시

## 요구사항
1. 회원가입 페이지
2. 로그인 페이지
3. 목표 추가 페이지
4. 목표 페이지 별 to do list화면
5. 현황 보여주는 graph 구현 (라이브러리 사용)
